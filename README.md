# starboard
bot.json
```
{
    "ignore_list": {},
    "bot": {
        "owner_id": int,
        "archive_channel": int,
        "archive_emote": str,
        "archive_emote_amount": int,
        "token": str
    }
}
```
```
"archive_emote": "<:emote_name:id>"
emote_name is the name you gave the emote
id is the emote id and can be obtained via "Open link"
```
example:
```json
{
    "ignore_list": {},
    "bot": {
        "owner_id": 212149701535989760,
        "archive_channel": 613348463341731883,
        "archive_emote": "<:LilyPad:544666310617858078>",
        "archive_emote_amount": 10,
        "token": "nein"
    }
}
```
